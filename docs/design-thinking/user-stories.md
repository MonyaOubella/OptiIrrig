# -------------------------------User Stories — OptiIrrig---------------------------------------------------#


 >>US-01 
 En tant qu'agriculteur je veux voir en temps réel l'humidité du sol et le débit afin de savoir si mes cultures sont bien irriguées 

 >>US-02  
 En tant qu'agriculteur je veux recevoir une recommandation automatique d'irrigation afin d'eviter le gaspillage d'eau 

 >>US-03  
En tant qu'agriculteur je veux etre alerté en cas de fuite avec localisation afin d'intervenir rapidement 

 >>US-04  
En tant qu'agriculteur je veux consulter l'historique de consommation afin de suivre mon évolution 

 >>US-05 
En tant que Resp. coopérative je veux voir toutes les exploitations sur un dashboard afin de détecter les anomalies globalement 

 >>US-06  
En tant que Resp. coopérative je veux exporter un rapport PDF mensuel afin de partager avec les partenaires 

 >>US-07 
En tant qu' agriculteur je veux Activer/désactiver la pompe à distance afin de contrôler l'irrigation sans se déplacer 

 >>US-08 
En tant qu'utilisateur je veux me connecter avec email/mot de passe afin de protéger mes données agricoles 




# ----------------------------------- MoSCoW — Priorisation des fonctionnalités -----------------------------#

## Must have 
- [US-01] Dashboard temps réel (humidité, débit d'eau)
- [US-02] Recommandation automatique d'irrigation (IA)
- [US-03] Alerte fuite avec localisation
- [US-08] Authentification sécurisée

## Should have 
- [US-04] Historique de consommation (semaine/mois)
- [US-07] Contrôle de la pompe à distance
- [US-05] Vue multi-exploitations pour coopératives

## Could have 
- [US-06] Export rapport PDF mensuel
- Notifications SMS en plus des alertes app
- Interface en langue arabe

## Won't have 
- Application mobile native (iOS/Android)
- Intégration météo en temps réel
- Marketplace de vente agricole
